# Sorairei NPC Maker
This is a fork of Sorairei NPC maker that aims to fix the problems with the original software. It loads all assets from a folder instead of having 50,000 .gifs. It also loads all NPCs directly from their folder and saves instantly.

## Installation
1.  Create an assets folder and add your `items.xml`, `items.otb`, `tibia.spr`, `tibia.dat` and `outfits.xml`
2.  Run `start.bat`
3. Load NPCs from your folder
